# Basic-Network-Scanning-with-Nmap
Basic Network Scanning with Nmap

Objective

The objective of this project is to perform network scanning using Nmap to identify open ports, running services, and the operating system of a local virtual machine. The project also analyzes the security implications of the discovered services.


What is Nmap?

Nmap (Network Mapper) is an open-source network scanning tool used by security professionals and system administrators to discover hosts, detect open ports, identify running services, and perform operating system detection.

It is widely used for:

Network Discovery
Security Auditing
Vulnerability Assessment
Asset Inventory
Service Detection


Why Network Scanning Matters

Network scanning helps administrators:

Discover active devices
Identify unnecessary open ports
Detect vulnerable services
Improve network security
Reduce attack surface

Regular scanning is an important part of cybersecurity.



Ethical Use

This project was performed only against a local virtual machine under my control.

Never scan systems that you do not own or do not have explicit permission to test. Unauthorized scanning may violate laws and organizational policies.
